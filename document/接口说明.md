# 接口说明

## 单条读取：LoadBizObject
- 地址：`https://www.h3yun.com/OpenApi/Invoke`
- 头部：`EngineCode`、`EngineSecret`
- 包体：`{"ActionName":"LoadBizObject","SchemaCode":"<表单编码>","BizObjectId":"<记录ID>"}`
- CLI：`python -m Web_Service/氚云/src/cli.py load-one --schema <SchemaCode> --id <BizObjectId>`

## 批量读取：LoadBizObjects
- 地址：`https://www.h3yun.com/OpenApi/Invoke`
- 头部：`EngineCode`、`EngineSecret`
- 包体包含：
  - `ActionName`=`LoadBizObjects`
  - `SchemaCode`：表单编码
  - `Filter`：字符串形式的 JSON（分页、返回字段、排序、查询条件）
- Filter 示例：`{"FromRowNum":0,"RequireCount":false,"ReturnItems":[],"SortByCollection":[],"ToRowNum":500,"Matcher":{"Type":"And","Matchers":[]}}`
- CLI（自动构建 Filter）：
  - 基本：`python -m Web_Service.氚云.src.cli load-many --schema <SchemaCode> --from 0 --to 500`
  - 条件：`python -m Web_Service.氚云.src.cli load-many --schema <SchemaCode> --matcher "{\"Type\":\"And\",\"Matchers\":[{\"Type\":\"Item\",\"Name\":\"Title\",\"Operator\":2,\"Value\":\"标题1\"}]}"`

## 单条新增：CreateBizObject
- 地址：`https://www.h3yun.com/OpenApi/Invoke`
- 头部：`EngineCode`、`EngineSecret`
- 包体包含：
  - `ActionName`=`CreateBizObject`
  - `SchemaCode`：表单编码
  - `BizObject`：字符串形式的 JSON，包含业务字段（如 `OwnerId`、`CreatedBy`、业务字段、子表等）
  - `IsSubmit`：`true`/`false`
- CLI：
  - 基本：`python -m Web_Service.氚云.src.cli create-one --schema <SchemaCode> --biz "{...}" --submit true`
  - `--biz` 既可传 JSON 字符串，也可传可解析的对象字符串（会自动转换为 JSON）

## 批量新增：CreateBizObjects
- 地址：`https://www.h3yun.com/OpenApi/Invoke`
- 头部：`EngineCode`、`EngineSecret`
- 包体包含：
  - `ActionName`=`CreateBizObjects`
  - `SchemaCode`：表单编码
  - `BizObjectArray`：字符串 JSON 数组，元素为每条记录的字符串 JSON
  - `IsSubmit`：`true`/`false`
- CLI：
  - 基本：`python -m Web_Service.氚云.src.cli create-many --schema <SchemaCode> --biz-array "[\"{...}\",\"{...}\"]" --submit true`
  - 或传对象数组：`--biz-array "[{...},{...}]"`（调用方会自动转换为字符串 JSON）
## 运算符说明（Matcher.Operator）
- `0` 大于
- `1` 大于等于
- `2` 等于
- `3` 小于等于
- `4` 小于
- `5` 不等于
- `6` 在范围内
- `7` 不在范围内
- `8` 包含（示例中的 `Like` 字段）
