# 氚云工具目录说明

```
Web_Service/氚云/
├─ src/         # 源代码：与氚云数据库读写相关的核心逻辑
├─ test/        # 测试：单元测试脚本、测试数据
├─ demo/        # 演示：可运行的示例脚本和用法演示
├─ document/    # 文档：使用说明、接口文档、环境要求
├─ i18n/        # 语言包：多语言翻译文件（示例 zh-CN、en-US）
├─ assets/      # 静态资源：图片/样式/模板等非代码资源
├─ plan/        # 计划：开发计划、需求文档、迭代规划
└─ changelog.md # 版本变更记录（语义化版本）
```

说明：新增目录均需在此文件补充说明，保持结构统一与可维护性。
