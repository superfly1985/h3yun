# 氚云工具目录说明

> 版本：v1.0.0
> 更新日期：2026-04-10

## 目录结构

```
01.氚云读写方法/
├── src/              # 核心源代码（Client、Config、CLI）
├── data/             # 数据操作脚本（独立可执行）
├── test/             # 测试文件和临时脚本
├── config/           # 配置文件（JSON格式）
├── document/         # 文档说明
├── demo/             # 演示示例
├── download/         # 下载文件存放
├── image/            # 图片资源
├── plan/             # 开发计划
├── assets/           # 静态资源
├── i18n/             # 国际化文件
├── .trae/rules/      # 项目规则
└── changelog.md      # 变更日志
```

## 详细说明

### src/ - 核心源代码
- 核心类定义（H3YunClient、H3YunConfig）
- CLI入口（cli.py）
- 可复用的工具函数
- **禁止**：测试脚本、临时脚本、配置文件

### data/ - 数据操作脚本
- 独立可执行的数据操作脚本
- 单功能脚本（读取、新增、更新、删除、上传、下载）
- 通过命令行参数接收输入

### test/ - 测试和临时脚本
- 单元测试脚本（`test_*.py`）
- 临时脚本（`tmp_*.py`，用完即删）
- 测试数据文件（JSON、txt）
- 环境配置示例（.env.example）

### config/ - 配置文件
- JSON格式的业务配置文件
- 上传配置、下载配置、过滤配置

### document/ - 文档
- 接口说明文档
- 使用指南
- 环境要求
- 操作经验

### demo/ - 演示示例
- 可运行的演示脚本
- 用法示例

### download/ - 下载文件
- 程序下载的文件
- 此目录内容不提交到版本控制

### image/ - 图片资源
- 文档配图
- 截图

### plan/ - 开发计划
- 开发计划文档
- 需求文档
