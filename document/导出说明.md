# 导出CSV使用说明

- 子命令：`export-csv`
- 参数：
  - `--schema` 表单编码（SchemaCode）
  - `--out` 输出CSV路径
  - `--from` 起始行（默认0）
  - `--to` 结束行（默认500）
  - `--matcher` 过滤条件（JSON字符串）或 `--matcher-file` 指定JSON文件路径
  - `--columns` 导出列（逗号分隔，不指定则自动推断并优先常用字段）
  - `--return-items` 返回字段数组（JSON字符串，默认`[]`）
  - `--env-file` 加载 `KEY=VALUE` 形式的环境变量文件（如 `H3YUN_ENGINE_CODE=...`）

- 示例：
  - 基本导出：
    - `python -m Web_Service.氚云.src.cli export-csv --schema D287764relabel --from 0 --to 50 --out Web_Service/氚云/test/export_sample.csv --columns ObjectId,Name,CreatedBy,OwnerDeptId,oldksk,newksk,IP,date`
  - 条件导出：
    - `python -m Web_Service.氚云.src.cli export-csv --schema D287764relabel --matcher-file Web_Service/氚云/test/matcher_submit.json --from 0 --to 10 --out Web_Service/氚云/test/export_match.csv`
